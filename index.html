Definitions

Data structures is defined differently by different people.

Definition 1:

A data structure is a named location that can be used to store and organize data. An algorithm is a finite set of instructions or logic, written in order,to accomplish a certain predefined task.

Definition 2:

A data structure is a way of collecting and organising data in such a way that we can perform operations on these data in an effective way.

Definition 3:

A data structure is data organization, management, and storage format that enables efficient access and modification.
